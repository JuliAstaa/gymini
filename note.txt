Penjelasan aplikasi
GYMINI adalah sebuah aplikasi yang kami buat, aplikasi ini bertipe manajemen karena mengelola aplikasi ini mengelola gym. Aplikasi ini memiliki database untuk menyimpan data data, seperti data member, user, membership pla, dan yang lainnya. Adapun collation pada table yang kami gunakan yaitu utf8mb4_0900_ai_ci, karena ini adalah default collation pada mysql versi 8.0.30. Untuk melakukan CRUD (Create, Read/Retrieve, Update, Delete) kami menggunakan menggunakan RESTful API yang dimana menggunakan HTTP request untuk mengirim dan mengelola data, untuk membuat API nya kami menggunakan bahasa pemrograman PHP. Untuk tampilan frontend atau dari sisi client itu sudah dinamis dengan menggunakan jQuery
